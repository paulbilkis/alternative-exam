Главная

Здравствуйте! Вашему вниманию предлагается тренажёр для тренировки поиска взаимно-однозначных соотношений между различными интерпретациями чисел Каталана.

Инструкция

Тренажёр имеет два режима: тренажёр и контроль. В обоих режимах программа выводит два столбца элементов для выбранной пары интерпретаций. У каждого элемента из левого столбца есть взаимно-однозначная пара в правом. Вам предлагается самостоятельно придумать правило, связывающее две интерпретации.
В режиме тренажёр вы можете проверить своё правило на простых примерах, для числа Каталана n=3.
В режиме контроль вам нужно будет применить ваше правило для значительно более сложных случаев, для чисел Каталана n=6
